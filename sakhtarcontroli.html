منظور از ساختار کنترلی چیست؟

یک ساختار کنترلی(Control Structure)به زبان ساده امکان کنترل گردش اجراي یک کد در اپلیکیشن را فراهم میکند. به طور کلی یک برنامه به طور ترتیبی و خط به خط اجرا میشود و ساختار کنترلی امکان تغییر دادن این گردش را ایجاد میکند. این تغییر گردش معمولاً از طریق بررسی یک شرط خاص صورت میگیرد. ساختارهاي کنترلی جزو ویژگیهاي اصلی زبان برنامهنویسی PHPهستند که به اسکریپتها امکان پاسخدهی متفاوت بسته به وروديها یا موقعیتهاي متفاوت را میدهند. بدین ترتیب اسکریپت میتواند بر اساس ورودي کاربر، محتواي یک فایل یا هر نوع داده دیگر پاسخ متفاوتی ایجاد کند.
فلوچارت زیر طرز کار ساختارهاي کنترلی در PHPرا نمایش میدهد.

همان طور که در نمودار فوق میبینید ابتدا یک شرط بررسی میشود. اگر شرط درست باشد، کد شرطی اجرا خواهد شد. نکته مهمی که باید توجه کنیم این است که پس از اجراي کد شرطی، اجراي کد به طور معمول ادامه مییابد. مثال زیر را در نظر بگیرید.

در مثال فوق برنامه بررسی میکند که آیا کاربر وارد شده است یا نه. بر اساس وضعیت ورود کاربر، برنامه وي را به صفحه Loginو یا My Accountهدایت میکند. در این مورد ساختار کنترلی با هدایت کاربر به صفحههاي متفاوت، اجراي خود را خاتمه میبخشد. این یکی از تواناییهاي اساسی زبان PHPمحسوب میشود.
PHPاز چند ساختار کنترلی پشتیبانی میکند:
  • if
  • else
  • elseif
  • switch
  • while
  • do-while
  • for
  • foreach
  • و موارد دیگر
در ادامه با معرفی مثالهایی، برخی از این ساختارها را بررسی میکنیم .در بخش قبل با مفاهیم مقدماتی ساختارهاي کنترلی در PHPو میزان مفید بودن آنها در توسعه اپلیکیشنها آشنا شدیم. در این بخش چند ساختار کنترلی مهم را معرفی میکنیم که به طور روزمره در توسعه اپلیکیشنها مورد استفاده هستند.

ساختارelse

در بخش قبلی ساختار ifرا بررسی کردیم که امکان اجراي یک بخش از کد در صورت درست بودن یک شرط را فراهم میسازد. در سوي دیگر اگر عبارتی نادرست باشد، آن ساختار کاري انجام نمیدهد. در اغلب موارد ما باید در صورت نادرست بودن یک شرط نیز کاري را اجرا کنیم. این همان جایی است که عبارت elseبه کار می آید.
عبارت elseهمواره همراه با ifاستفاده میشود. اساس کار این عبارت در شبه کد زیر مشخص شده است : 
  
            if (expression)
            {
            // code is executed if the expression evaluates to TRUE
            }
            else
            {
            // code is executed if the expression evaluates to FALSE
            }

ساختار اساسی عبارت elseifرا میتوانید در شبه کد زیر بررسی کنید :

            {
            // code is executed if the expression1 evaluates to TRUE
            }
            elseif (expression2)
            {
            // code is executed if the expression2 evaluates to TRUE
            } 
            elseif (expression3)
            {
            // code is executed if the expression3 evaluates to TRUE
            }
            else
            {
            // code is executed if the expression1, expression2 and expression3
            evaluates to FALSE, a default choice
            }

        while (expression)
        { 
            // code to execute as long as expression evaluates to TRUE
        }

        do
        {
        // code to execute
        } while (expression);
